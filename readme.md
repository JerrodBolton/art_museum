# Art Museum Assignment

## Prerequisites
- A clear understanding of different types of art museums
- Online visits to art museums around the world that interest you

## Skills to Master
- Understanding art museums
- Applying AI to achieve hyper-productivity in art creation

## Overview

## Guidelines
- Pick themes that interest you
- No expertise required in your chosen subject
- Attend demos on building online art museums

## Requirements
- Choose compelling themes for your museum
- Generate 3-10 art images using generative AI
- Have AI code an `index.html` file
- Store all images in the same folder as `index.html`
- Use concise, meaningful image filenames
- Keep each image under 25 MB (use MS Paint to resize if needed)
- Title each artwork with a description and tool attribution
- Create a GitHub repository and upload files
- Deploy as a GitHub Pages website with a shareable HTTP link

## Deliverables
- Share your GitHub Pages URL (`.github.io` domain)
- **Do not submit the raw GitHub file URL**
